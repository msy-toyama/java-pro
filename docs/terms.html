1適用

本利用規約（以下「本規約」といいます）は、Masaya Kobayashi（以下「開発者」といいます）が提供する iOS アプリケーション「Java Pro」（以下「本アプリ」といいます）の利用条件を定めるものです。

ユーザーは、本アプリをダウンロード、インストール、または使用することにより、本規約のすべての条項に同意したものとみなされます。本規約に同意いただけない場合は、本アプリの使用をお控えください。

2サービスの内容

本アプリは、Java プログラミング言語の学習および Oracle Java 認定資格（SE 11 / SE 17 の Silver / Gold）の取得に向けた学習を支援する iOS アプリケーションです。以下の機能を提供します。

  1. 体系的な Java 学習コンテンツ（全34コース・169レッスン）の閲覧
  2. 多様なクイズ形式による理解度確認（四択・複数選択・穴埋め・並び替え・出力予測・エラー発見・コード補完・試験形式の全8種、計575問）
  3. 実践演習（39チャプター・108問）および Java 環境構築ガイド（5セクション）
  4. 非公式模擬試験の受験（SE11/SE17 × Silver/Gold × 各2セット = 計8種・640問）
  5. 学習進捗の管理およびゲーミフィケーション（XP・バッジ・レベル・連続学習日数）
  6. 弱点分析・資格別進捗トラッキング
  7. 間隔反復法に基づく復習システム（4段階：即時→ 24時間 → 3日 → 7日 → 習得完了）
  8. Java 用語集（202語）の閲覧・検索
  9. 学習リマインダー通知（ローカル通知）
  10. 初回利用時のガイドツアー（各画面の使い方案内）

本アプリは Java 学習を支援するツールであり、資格試験の合格を保証するものではありません。

3無料範囲と課金

  1. 無料コンテンツ: 本アプリは無料でダウンロードできます。入門コース（Beginner 8コース・46レッスンの全レッスン・クイズ）、SE11 Silver 模擬試験1、用語集、バッジ・レベルシステム、復習機能、環境構築ガイドは無料でご利用いただけます。
  2. フルアクセス（買い切り ¥980）: Silver / Gold 範囲のコース、残り7種の模擬試験、および有料実践演習をご利用いただくには、アプリ内課金（買い切り）による「フルアクセス」の購入が必要です。サブスクリプション（月額・年額課金）ではありません。
  3. 決済: 購入は Apple の App Store 決済を通じて行われます。課金・返金については Apple メディアサービス利用規約 に準拠します。
  4. 購入の復元: 機種変更やアプリの再インストール時は、設定画面の「購入を復元」ボタンから購入済みの権利を復元できます。
