## 1. プロジェクト概要

### 1.1 目的

プロプロ（プログラミングのプロになる -Java入門-）は、Oracle Java 認定資格（SE 11 / SE 17 の Silver / Gold）の取得を目指す日本語話者向けの iOS 学習アプリケーションである。iOS インストール時の表示名は「プロプロ」。初心者から上級者まで段階的に学習でき、ゲーミフィケーション要素により学習継続を促進する。テキスト学習・クイズ・実践演習・模擬試験を 1 つのアプリ内で完結させ、オフラインで利用可能とする。

### 1.2 対象ユーザー

| ペルソナ | 概要 |
|---|---|
| IT 業界新卒 | 企業から Java Silver 取得を求められている 22〜25 歳 |
| キャリアチェンジ | プログラミング学習中の 25〜35 歳。通勤時間に学習したい |
| Java Gold 挑戦者 | Silver 取得済みで Gold を目指す中堅エンジニア |

### 1.3 対象プラットフォーム

| 項目 | 仕様 |
|---|---|
| OS | iOS 18.0 以上 / iPadOS 18.0 以上 |
| デバイス | iPhone / iPad（ユニバーサル） |
| 配信 | App Store |
| 言語 | 日本語 / 英語（アプリ内切替） |

## 9. 用語定義

| 用語 | 定義 |
|---|---|
| チャプター / コース | 特定のテーマに関するレッスン群をまとめた単位 |
| レッスン | 1 つの学習トピックを扱う最小の教材単位（3〜5 分で完了） |
| クイズ | レッスンの理解度を確認するための問題 |
| 模擬試験 | Oracle 認定資格の本番試験を模した実践的なテスト |
| 実践演習 | Java コードを実際に書いて動かす練習問題 |
| XP | 学習アクションに対して付与される経験値 |
| バッジ | 特定の条件を達成した際に獲得できる称号 |
| ストリーク | 連続で学習した日数 |
| 間隔反復 | 忘却曲線に基づき最適な間隔で復習を行う学習法 |
| フルアクセス | 有料購入により解放されるすべてのコンテンツ・機能 |
| ガイドツアー | 各画面への初回訪問時に表示されるステップバイステップの使い方案内 |
